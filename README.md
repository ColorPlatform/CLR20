## Tests

##### 1. Install [Truffle](https://truffleframework.com/)
```
npm install -g truffle
```
##### 2. Launch Ethereum Client ([Ganache](https://truffleframework.com/ganache))
##### 3. Run javascript test files:
```
truffle test
```
##### 4. Check all 128 tests have passed
